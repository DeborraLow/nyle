# Nyle

'Nyle' is a minimal graphics framework using Ruby/GTK3 and Cairo.

## Requirements

* Ruby/GTK3
* rsvg2 (for Windows only)

## Installation

Install it yourself as:

    $ gem install nyle

Or for Windows

    $ gem install nyle
    $ gem rsvg2

## Usage

```ruby
require 'nyle'
```
